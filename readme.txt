To run the game:

Run an instance of server.py, this can be run on any machine on your local network.
Make sure the IP address is set to the local IP of the machine it is being run on.

Edit network.py to hold the same IP as you listed in server.

Make sure fighter.py, network.py, move.py, game.py and the images folder are in the same directory on the device you want to run the game on.

Run two instances of game.py, make sure active_player is set to 1 on one device playing and 0 on the other.
